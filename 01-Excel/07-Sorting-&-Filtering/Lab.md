# 📘 Lab – Sorting & Filtering in Microsoft Excel

## 🎯 Objective

The objective of this lab is to gain practical experience with **Sorting** and **Filtering** features in Microsoft Excel using a real-world employee dataset.

---

# 📂 Dataset Required

**Employee_Sorting_Filtering_Dataset.xlsx**

