# Interview Questions

# Excel Interface 

## 1. What is Microsoft Excel?

**Answer**

Microsoft Excel is a spreadsheet application used to store, organize, clean, analyze, and visualize data using rows, columns, formulas, functions, charts, and PivotTables.

---

## 2. What is a Workbook?

**Answer**

A workbook is an Excel file that contains one or more worksheets.

---

## 3. What is a Worksheet?

**Answer**

A worksheet is a single spreadsheet inside a workbook where data is stored in rows and columns.

---

## 4. What is a Cell?

**Answer**

A cell is the intersection of a row and a column where data is entered.

---

## 5. What is the Name Box?

**Answer**

The Name Box displays the address of the currently selected cell and can also be used to quickly navigate to another cell by entering its reference.

---

## 6. What is the Formula Bar?

**Answer**

The Formula Bar displays and allows you to edit the content or formula of the selected cell.

---

## 7. Which Ribbon tab is mainly used for sorting and filtering?

**Answer**

The **Data** tab.

---

## 8. Which Ribbon tab is used to insert charts?

**Answer**

The **Insert** tab.

---

# Importing Data

## 1. What is data import in Excel?

**Answer:**

Data import is the process of bringing data from external sources into Excel for analysis and reporting.

---

## 2. What does CSV stand for?

**Answer:**

CSV stands for **Comma-Separated Values**.

---

## 3. Which Excel tab is mainly used to import data?

**Answer:**

The **Data** tab.

---

## 4. Name some sources from which Excel can import data.

**Answer:**

- Excel Workbook
- CSV
- Text File
- PDF
- Web
- SQL Server
- Microsoft Access
- JSON
- XML

---

## 5. Why is importing data better than manual entry?

**Answer:**

- Saves time
- Reduces errors
- Supports large datasets
- Increases efficiency
- Makes analysis easier

---

# Data Formatting

## 1. What is data formatting?

**Answer:**

Data formatting changes the appearance of data without changing its actual value.

---

## 2. Why is formatting important in Excel?

**Answer:**

Formatting improves readability, consistency, professionalism, and helps reduce errors during analysis.

---

## 3. What is Wrap Text?

**Answer:**

Wrap Text displays long text on multiple lines within the same cell.

---

## 4. What does Format Painter do?

**Answer:**

Format Painter copies formatting from one cell or range and applies it to another.

---

## 5. What is the difference between Currency and Accounting formats?

**Answer:**

- **Currency** places the currency symbol directly next to the number.
- **Accounting** aligns currency symbols and decimal points, making financial reports easier to read.

---

## 6. Why should merged cells be avoided in datasets?

**Answer:**

Merged cells can interfere with sorting, filtering, formulas, and PivotTables. They are best avoided in data tables.
