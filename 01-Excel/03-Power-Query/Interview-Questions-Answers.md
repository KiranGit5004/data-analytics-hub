# Interview Questions

## 1. What is Power Query?

**Answer**

Power Query is an ETL (Extract, Transform, Load) tool in Microsoft Excel and Power BI used to import, clean, transform, and load data from various sources.

---

## 2. What does ETL stand for?

**Answer**

- Extract
- Transform
- Load

---

## 3. Why is Power Query better than manual data cleaning?

**Answer**

Because it automates repetitive tasks, reduces manual effort, minimizes errors, and allows data to be refreshed automatically.

---

## 4. What is the purpose of Applied Steps?

**Answer**

Applied Steps records every transformation performed in Power Query, allowing the process to be repeated automatically whenever the data is refreshed.

---

## 5. Can Power Query connect to databases?

**Answer**

Yes. Power Query supports SQL Server, MySQL, PostgreSQL, Oracle, Access, SharePoint, Web sources, and many other data sources.

---

## 6. Does Power Query modify the original data source?

**Answer**

No. Power Query works on a copy of the data and leaves the original source unchanged.
