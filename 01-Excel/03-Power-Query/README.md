# Power Query

This folder contains notes, hands-on labs, and interview questions for **Microsoft Excel Power Query**. It covers the fundamentals of importing, cleaning, transforming, and preparing data for analysis.

## Contents

- **Power Query Notes** – Core concepts and transformation techniques.
- **Power Query Lab** – Hands-on exercises with step-by-step instructions.
- **Interview Questions & Answers** – Common Power Query interview questions for practice.

## Topics Covered

- Introduction to Power Query
- Importing Data
- Data Cleaning
- Removing Duplicates
- Handling Missing Values
- Fill Up / Fill Down
- Trim and Clean Text
- Removing Rows and Columns
- Data Type Conversion
- Custom Columns
- Loading Data to Excel

## Learning Outcomes

After completing this module, you will be able to:

- Import data from multiple sources.
- Clean and transform datasets using Power Query.
- Create custom columns and perform calculations.
- Prepare data for reporting and analysis.
- Answer basic Power Query interview questions.
