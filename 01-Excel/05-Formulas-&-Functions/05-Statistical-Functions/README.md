# 📌 Best Practices

- Use **AVERAGEIFS()** when multiple conditions are required.
- Use **MEDIAN()** instead of AVERAGE() when data contains outliers.
- Use **MODE.SNGL()** to identify the most frequent value.
- Use **LARGE()** and **SMALL()** for Top-N and Bottom-N analysis.
- Use **RANK.EQ()** to rank records without sorting the dataset.
- Use **PERCENTILE.INC()** and **QUARTILE.INC()** to understand data distribution.
- Use **STDEV.S()** when working with sample data.
