# 📘 Lookup & Reference Functions

Welcome to the **Lookup & Reference Functions** module of my **Data Analytics Hub**.

Lookup & Reference Functions are among the most important Excel functions for **Data Analysts**, **Business Analysts**, **MIS Executives**, and **Reporting Professionals**. These functions help retrieve, match, and combine data from multiple tables, making them essential for working with real-world business datasets.

This module provides detailed notes, hands-on labs, interview questions, practical solutions, and multiple datasets to help you master Lookup & Reference Functions.

# 📌 Best Practices

- Use **XLOOKUP()** whenever available.
- Use **INDEX + MATCH()** for flexible and scalable solutions.
- Avoid hardcoded column numbers.
- Always use **FALSE** for exact matches in VLOOKUP.
- Use **IFERROR()** to display meaningful messages.
- Keep lookup tables clean and free from duplicate keys.
- Lock lookup ranges using absolute references (`$A$2:$D$100`).
