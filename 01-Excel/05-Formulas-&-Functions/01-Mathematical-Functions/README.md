# 📌 Best Practices

- Use cell references instead of hard-coded values.
- Verify that numeric data is stored as **Number**, not **Text**.
- Use ranges instead of selecting individual cells whenever possible.
- Test formulas using sample data before applying them to large datasets.
- Keep formulas simple, readable, and easy to maintain.
