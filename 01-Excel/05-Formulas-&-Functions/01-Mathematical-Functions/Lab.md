# 📘 Lab 01 – Mathematical Functions

## Objective
Practice Excel mathematical functions using a realistic employee dataset.

## Dataset
**File:** `Employee_Dataset_200.xlsx`

## Functions Covered
- SUM
- AVERAGE
- MIN
- MAX
- COUNT
- COUNTA
- COUNTBLANK
- SUMIF
- SUMIFS
- ROUND
- ROUNDUP
- ROUNDDOWN
- ABS
- INT
- MOD
- QUOTIENT
- CEILING
- FLOOR

## Tasks

### Part A – Basic Calculations
1. Calculate Total Salary.
2. Calculate Average Salary.
3. Find Highest Salary.
4. Find Lowest Salary.
5. Count numeric Salary records.
6. Count non-empty Employee Names.
7. Count blank cells in Bonus.

### Part B – Conditional Calculations
8. Total Salary for IT department.
9. Total Salary for IT department in Mumbai.

### Part C – Rounding
10. Monthly Salary = Salary/12 using ROUND().
11. Monthly Salary using ROUNDUP().
12. Monthly Salary using ROUNDDOWN().

### Part D – Other Functions
13. ABS(-250)
14. INT(15.98)
15. MOD(Salary,1000)
16. QUOTIENT(Salary,1000)
17. CEILING(Salary,5000)
18. FLOOR(Salary,5000)

## Challenge
- Department-wise total salary.
- City-wise average bonus.
- Employees earning above average salary.
- Build a summary table with Total, Average, Max and Min Salary.

## Learning Outcome
Use mathematical functions confidently on business datasets.
