# Excel Text Functions

## 📖 Overview

Text Functions are among the most frequently used functions in Microsoft Excel. They help Data Analysts clean inconsistent data, extract meaningful information, standardize text, and prepare datasets for reporting and analysis.

In real-world projects, data imported from websites, databases, ERP systems, CRM software, or CSV files often contains formatting issues such as extra spaces, inconsistent capitalization, text stored as numbers, or combined values. This module teaches you how to solve these problems efficiently.

# 📚 Functions Covered

| Function | Purpose |
|----------|----------|
| LEFT() | Extract characters from the left side |
| RIGHT() | Extract characters from the right side |
| MID() | Extract characters from the middle |
| LEN() | Count total characters |
| TRIM() | Remove extra spaces |
| UPPER() | Convert text to uppercase |
| LOWER() | Convert text to lowercase |
| PROPER() | Capitalize each word |
| CONCAT() | Combine text values |
| TEXTJOIN() | Combine text using a delimiter |
| TEXT() | Format numbers as text |
| VALUE() | Convert text to numbers |
| FIND() | Find text position (case-sensitive) |
| SEARCH() | Find text position (case-insensitive) |
| REPLACE() | Replace characters by position |
| SUBSTITUTE() | Replace matching text |

# 💡 Best Practices

- Always clean imported data before analysis.
- Use `TRIM()` after importing CSV or web data.
- Store email addresses in lowercase.
- Use `PROPER()` for names.
- Prefer `TEXTJOIN()` when separators are required.
- Use `VALUE()` to convert imported numbers.
- Combine multiple functions to solve complex problems.
- Test formulas on sample data before applying them to large datasets.

---

# ⚠ Common Mistakes

- Forgetting to remove extra spaces.
- Confusing `FIND()` with `SEARCH()`.
- Using `REPLACE()` instead of `SUBSTITUTE()`.
- Performing calculations on text values.
- Forgetting spaces while using `CONCAT()`.
- Ignoring text formatting before creating reports.
