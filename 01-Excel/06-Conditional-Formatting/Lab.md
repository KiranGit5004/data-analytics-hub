# 📘 Lab – Conditional Formatting in Microsoft Excel

## 🎯 Objective

The objective of this lab is to gain hands-on experience with **Conditional Formatting** in Microsoft Excel using real-world business data.

By completing this lab, you will learn how to:

- Highlight important values automatically.
- Identify duplicate records.
- Detect missing information.
- Compare business performance.
- Visualize trends using colors and icons.
- Create dynamic reports.
