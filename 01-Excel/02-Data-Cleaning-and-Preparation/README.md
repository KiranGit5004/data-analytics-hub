# Data Cleaning and Preparation

This folder contains notes and interview questions related to **Data Cleaning and Preparation** in Microsoft Excel.

## Topics Covered

- Removing Duplicates
- Handling Missing Values
- Standardizing Data Formats
- Interview Questions & Answers

## Learning Outcomes

After completing these notes, you will be able to:

- Clean messy datasets
- Remove duplicate records
- Handle missing values effectively
- Standardize dates, numbers, and text formats
- Prepare data for analysis
- Answer basic interview questions on data cleaning
